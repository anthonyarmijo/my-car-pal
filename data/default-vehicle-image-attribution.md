# Default Vehicle Image Attribution

This catalog uses transparent PNG cutouts derived from reviewed Wikimedia Commons
images. Original photos had backgrounds removed via BiRefNet (birefnet-general model)
processing, then resized to 1600px wide. The app serves only local cutout assets —
no runtime hotlinking to third-party image hosts.

**Image requirements:** All catalog images must be front-facing or front 3/4 view.
Rear-angle shots are not accepted for make/model defaults.

Keep each imported image paired with source URL, author/attribution, license,
license URL, and commercial-use confirmation in `data/default-vehicle-images.json`.

## Cutout Processing Notes

- **Tool:** rembg 2.0.75 with BiRefNet (birefnet-general model) — automated background removal
- **Resize target:** 1600px wide (proportional height), web-optimized PNG
- **Source license:** all originals are CC BY-SA 4.0 (commercial use allowed with attribution)
- **Derivative status:** cutouts are derivative works under the same CC BY-SA 4.0 license
- **View requirement:** front or front 3/4 view only — no rear-angle shots
- **Known issues:**
  - Honda CR-V: UK-spec model (right-hand drive), acceptable for default catalog image
  - GMC Sierra 1500: X31 off-road trim (grille differs from base models)
